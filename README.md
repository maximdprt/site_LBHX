# Lacanau Beach Handball Xperience (LBHX)

Site one-page pour l'événement LBHX 2026 — Du 12 au 14 juin à Lacanau Océan.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Installation

```bash
npm install
npm run dev
```

Ouvre [http://localhost:3000](http://localhost:3000).

## Vidéo teaser

La vidéo `Teaser LBHX.mp4` (130 Mo) dépasse la limite GitHub de 100 Mo et n'est pas dans le repo.

**Après clonage**, placez le fichier ici :
```
public/videos/Teaser LBHX.mp4
```

Elle servira de fond vidéo dans la section Hero.

## Build

```bash
npm run build
```

Génère un site statique dans le dossier `out/` (déploiement simple sur tout hébergeur statique).
